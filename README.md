Sell Price Multiplier
A BepInEx plugin that allows configuation of the base sell multiplier.


Installation
Install BepInEx into your game directory.
Place sellmultiplier.dll into BepInEx/plugins/.
Run the game once to generate the config file at BepInEx/config/com.goblinvyke.traderpricemulti.cfg.
Edit the config to set your desired price multiplier either with the cfg file generated or BepInEx Configuration Manager in game with F1(Default).

Configuration
Setting	Default	Range	Description
SellMultiplier =	0.1 min – 10 max	Multiplier applied to trader prices
